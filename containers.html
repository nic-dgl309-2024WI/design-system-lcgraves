05 Containers

I. Flex II. Grid III. Max Width IV. Full Width V. Examples

I. Flex layout

Functionality Class Name Description
Flex Container l-flex Set display to flex.
l-flex-row Set flex direction to row.
l-flex-column Set flex direction to column.
Flex Wrap l-flex-wrap Set flex items to wrap.
Justify Content l-flex-justify-start Justify content to the start of the flex container.
l-flex-justify-end Justify content to the end of the flex container.
l-flex-justify-center Justify content to the center of the flex container.
l-flex-justify-between Justify content with space between the items.
l-flex-justify-around Justify content with space around the items.
l-flex-justify-evenly Justify content with space evenly distributed.
Align Items l-flex-align-start Align items to the start of the cross axis.
.l-flex-align-center Align items to the center of the cross axis.
l-flex-align-end Align items to the end of the cross axis.

II. Grid layout

Functionality Class Name Description
Grid Container l-grid Set display to grid.
Grid Columns l-grid-cols-2 Define 2 columns.
l-grid-cols-3 Define 3 columns.
l-grid-cols-4 Define 4 columns.
Grid Rows l-grid-rows-2 Define 2 rows.
l-grid-rows-3 Define 3 rows.
l-grid-rows-4 Define 4 rows.
Grid Gap l-grid-gap-1 Set gap to 1rem.
l-grid-gap-1-5 Set gap to 1.5rem.
l-grid-gap-2 Set gap to 2rem.
l-grid-gap-3 Set gap to 3rem.
Grid Gap Columns l-grid-gap-column-1-5 Set column gap to 1.5rem.
Justify Items l-grid-justify-items-start Justify items to start.
l-grid-justify-items-center Justify items to center.
Align Items .l-grid-align-items-start Align items to start.
l-grid-align-items-center Align items to center.
Justify Content l-grid-justify-content-start Justify content to start.
l-grid-justify-content-center Justify content to center.
Align Content l-grid-align-content-start Align content to start.
.l-grid-align-content-center Align content to center.
Align Self .l-grid-align-self-start Align self to start (individual item).
l-grid-align-self-end Align self to end (individual item).
Justify Self l-grid-justify-self-end Justify self to end (individual item).

III. Max width

Class Name Description
width-90 Container width set to 90% of the parent element, centered with auto margins.
width-90--desktop Container max-width set to 90% of its parent element, including padding and centered.
width-80 Container with max-width set to 80% of the parent, centered, and includes padding.
text-wrapper Container designed for text, with a max and min width for optimal reading and centered.
width-630-center--desktop Container with a max-width of 630px on desktop, centered, and includes padding.
width-325-550 Container with a minimum width of 325px and a maximum width of 550px.

IV. Full width

Class Name Description
full-width Full-width container with auto margins to center it.
